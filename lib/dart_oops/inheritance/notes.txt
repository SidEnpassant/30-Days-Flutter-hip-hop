child class inherit , use methods or properties from parent class , Inheritance in dart is a key feature
of Object Oriented Programming that allows a class (known as a subclass or derived class) to inherit properties
and methods from another class (known as a superclass or base class). This enables code reusability and establishes
a natural hierarchy between classes.

Key Concepts of Inheritance in Dart
1. Superclass and Subclass:
   Ο Superclass base parent: The class whose properties and methods are inherited.
   Ο Subclass derived: The class that inherits properties and methods from the superclass.
2. extends Keyword:
   Ο In Dart, the extends keyword is used to indicate that one class is inheriting from another class.
3. Access to Superclass Members:
   Ο The subclass can access public and protected properties and methods of the superclass.
   Ο Private members (declared with a leading underscore) are not accessible outside the file they are declared in.
4. Overriding Methods:
   Ο Subclasses can override methods of the superclass to provide specific implementations using the @overrideannotation
5. Constructors:
   Ο Constructors of the superclass are not inherited by the subclass , but the subclass can call the superclass
   constructor using the super keyword.
